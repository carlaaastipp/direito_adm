# direito_adm
Estudar Direito Administrativo de forma simplificada envolve resumos esquematizados, videoaulas curtas e PDFs gratuitos que focam em conceitos chave como princípios e atos administrativos, sem aprofundar em doutrinas complexas. Essa abordagem é ideal para iniciantes ou revisões rápidas em concursos, priorizando tabelas e mapas mentais.
